# App Template

Template for creating new SaaS applications on GSV Platform.

## Quick Start

```bash
# Copy this template
cp -r apps/app-template apps/my-new-app
cd apps/my-new-app

# Update package.json name
# Update README.md

# Install and run
yarn install
yarn dev
```

## Structure

- Standard Next.js 14 app
- TypeScript
- Ready for deployment via GitOps
