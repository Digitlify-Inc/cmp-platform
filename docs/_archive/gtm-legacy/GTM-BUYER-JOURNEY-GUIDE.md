# Buyer Journey - Step-by-Step Guide

**Last Updated:** 2025-12-16

## Overview

Browse Catalog -> Purchase -> Configure -> Get Keys -> Use

## Step 1: Browse Marketplace

1. Go to https://app.dev.gsv.dev
2. Login with SSO
3. Click Marketplace
4. Browse: Agents, Apps, Assistants, Automations

## Step 2: Purchase

1. Click offering
2. Select plan
3. Subscribe
4. Complete payment

## Step 3: Configure Agent

- Persona: name, greeting, prompt, tone
- Branding: color, avatar
- Widget: position, theme, domains

## Step 4: Get API Keys

1. API Keys tab
2. Create key
3. Copy: ar_sk_live_xxxxxxxx

## Step 5: Use

Widget: widget.digitlify.com/loader.js
API: POST /api/agent-gateway/invoke/
