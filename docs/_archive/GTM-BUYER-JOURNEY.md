| Action | Credits | Notes |
|--------|---------|-------|
| Try (demo mode) | 0 | Limited to 5 messages |
| Subscribe | 0 | Subscription is free |
| Chat message | 1-3 | Depends on agent type |
| RAG document upload | 5 | Per document processed |
| Image generation | 10 | If capability enabled |
| Voice interaction | 2 | If capability enabled |

## Tenant Isolation

Each subscription is isolated by:
1. **Project scope** - Subscriptions belong to specific projects
2. **Langflow workspace** - Each project maps to Langflow workspace
3. **Ragflow collection** - Each subscription has own vector collection
4. **API keys** - Scoped to subscription, not shared

See [GTM-TENANT-ISOLATION.md](./GTM-TENANT-ISOLATION.md) for detailed isolation model.
