# Capability Page — Web Widget + Branding v1
_Last updated: 2025-12-19_

> Route: `/capabilities/web_widget_branding`

## What it enables
Deploy an assistant on your website with your branding:
- colors, logo, tone, greetings
- allowed domains + CORS protection
- widget sessions issued by Gateway

## When you need it
- customer support assistant
- onboarding assistant
- lead capture assistant

## What’s configurable (examples)
- theme (primary/secondary color)
- widget position
- welcome message
- escalation (handoff email/slack channel)
- languages enabled

## Pricing notes
- Runs consume credits like any other execution
- Widget sessions are free; executions are metered
