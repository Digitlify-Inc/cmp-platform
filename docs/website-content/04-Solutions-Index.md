## Role page template (6 Golden Questions)
Use this layout for `/solutions/{role}`.

1) **WHY**: pain + stakes  
2) **WHAT**: the measurable outcome  
3) **WHO**: persona + stack  
4) **HOW**: connect → configure → run/deploy  
5) **HOW MUCH**: credits + recommended plan  
6) **WHAT NEXT**: adjacent bundles + capabilities
