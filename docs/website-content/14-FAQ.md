# FAQ Page v1
_Last updated: 2025-12-19_

## Getting started
**Do I need to code?** No — start with bundles and templates.

**How fast can I see value?** Same day for simple bundles; 1–2 weeks for deeper integrations.

## Credits
**What are credits?** Credits are your budget for AI work. Each run consumes credits based on model/tools/workload.

**What happens when I run out?** Runs pause and you can top up instantly.

## Security
**How are secrets handled?** Vault + ESO; secrets are never stored in templates.

**Can tenants see each other’s data?** No — hard isolation boundaries are enforced.
